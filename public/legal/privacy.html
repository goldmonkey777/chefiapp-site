ChefIApp™

Política de Privacidade

Última atualização: Dezembro 2025

O ChefIApp™ é operado pela Goldmonkey Studio LLC, uma LLC registrada em Wyoming, EUA, com base operacional em Ibiza, Espanha ("nós", "nosso", "nos").

1. Dados que recolhemos

  • Dados de conta: nome, email, idioma, fotografia opcional.
  • Dados de uso: tarefas concluídas, check-ins, pontos de XP, métricas de desempenho.
  • Dados de localização aproximada: utilizados apenas para validar check-ins de turno.
  • Identificadores técnicos: ID do dispositivo, logs técnicos básicos.

2. Como utilizamos os dados

  • Para operar o ChefIApp™ e disponibilizar as funcionalidades do serviço.
  • Para criar dashboards, relatórios e métricas internas do restaurante.
  • Para melhorar o produto, de forma agregada e anónima sempre que possível.
  • Para comunicação operacional e suporte.

3. Partilha de dados

Não vendemos dados pessoais. Podemos partilhar dados apenas com:

  • Prestadores de serviço estritamente necessários (ex.: Supabase, serviços de cloud).
  • Autoridades legais, quando exigido por lei.

4. Retenção e segurança

Retemos dados apenas pelo tempo necessário para operar o serviço ou cumprir obrigações legais. Utilizamos encriptação em trânsito (HTTPS/TLS) e boas práticas de segurança de aplicação.

5. Direitos do utilizador

Dependendo da jurisdição, você pode ter direito a aceder, corrigir ou eliminar os seus dados. Para exercer estes direitos, contacte-nos em contact@goldmonkey.studio.
